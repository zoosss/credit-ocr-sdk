## 覆盖范围

🏦 **支持 4300+ 家银行，15000+ 种流水样式**

覆盖六大国有银行（工农中建交邮储）、12 家全国性股份制银行、以及绝大部分城商行和农商行。

支持格式：纸质流水图片、PDF 电子流水、Excel 电子流水（xls/xlsx/csv）

## 分析维度

流水分析报告包含以下核心维度：

| 分析维度 | 分析内容 |
|---------|---------|
| 收入分析 | 月均收入、工资收入、经营收入、其他收入占比、收入稳定性 |
| 支出分析 | 月均支出、固定支出、消费支出、大额异常支出标记 |
| 资金流向 | 高频交易对手识别、关联账户分析、循环转账检测 |
| 异常检测 | 刷流水识别、夜间大额交易、快进快出、余额异常波动 |
| 还款能力 | 负债收入比、可用余额趋势、结余率、综合评分 |
| 真伪鉴别 | PS 篡改痕迹检测、自动拦截假流水 |
