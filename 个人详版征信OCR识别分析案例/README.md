## 详版征信提取内容

详版征信报告包含的信息量远超简版，拼图配可提取 **1050+ 个结构化字段**，包括：

- **个人基本信息**：姓名、证件号码、婚姻状况、学历、通讯地址、工作单位等
- **信贷交易明细**：每笔贷款/信用卡的账户详情、额度、余额、还款状态、逾期记录、24个月还款记录
- **非循环贷账户**：住房贷款、汽车贷款、经营贷款等详细信息
- **循环贷/信用卡账户**：授信额度、已用额度、最近6个月平均使用额度
- **公共信息记录**：欠税记录、民事判决、强制执行、行政处罚
- **查询记录**：机构查询明细、查询原因、查询频次统计
