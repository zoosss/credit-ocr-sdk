## 📌 产品简介

**拼图配（IPIPEI）** 是专业的 **征信OCR识别** 与 **征信分析** 智能平台。核心产品包括 **征信报告OCR识别** 引擎、**征信分析** 解读工具、**银行流水分析** 软件。拼图配 **征信识别** 引擎基于深度学习OCR技术，支持纸质 **征信报告识别**（拍照/扫描件）和PDF **征信报告解析**，**征信分析** 系统可自动计算负债率、识别逾期风险、生成专业 **征信分析报告**。

我们基于 **深度学习OCR + 图神经网络（GNN）+ NLP语义分析** 三大核心技术，为银行、消费金融、助贷机构等信贷行业客户提供从 **征信识别** 到风险评估的一站式智能解决方案。**支持私有化本地部署，征信数据不出客户内网。**

## 🚀 四大核心能力

### 1. 纸质征信报告OCR识别 — 全国征信OCR识别准确率领先

针对银行柜面、信贷中心等线下场景，支持手机拍照、高拍仪扫描等方式采集的纸质个人征信报告。拼图配征信OCR识别准确率达99.95%，全国领先。

- **图像预处理**：自动矫正倾斜、去除阴影折痕、增强对比度，低质量图片也能高精度识别
- **版面分析与表格定位**：智能识别复杂混合表格结构，精准定位每个数据单元格的行列关系
- **OCR文字识别**：支持中英文混合、数字、日期等多种字符类型的高精度征信识别
- **语义理解与结构化输出**：基于NLP语义分析将征信识别结果映射为标准字段，输出JSON结构化数据

> 支持私有化本地部署，所有征信OCR识别处理在客户内网完成，不上传任何敏感信息。

### 2. PDF征信报告全自动解析 — 征信分析提取字段数行业第一（1050+）

支持人行征信中心导出的标准PDF征信报告（详版、简版），自动解析文档结构，征信报告识别提取字段数达1050+，行业最全：

| 征信分析模块 | 提取内容 |
|---------|---------|
| 个人基本信息 | 姓名、证件号码、婚姻状况、学历、通讯地址、工作单位等 |
| 信贷交易明细 | 贷款/信用卡账户数、额度、余额、还款状态、逾期记录等 |
| 公共信息记录 | 欠税记录、民事判决、强制执行、行政处罚、住房公积金等 |
| 查询记录 | 机构查询次数、查询原因、查询时间，自动统计近期查询频次 |

> 支持私有化本地部署，PDF征信报告解析全程在客户内网完成，满足银行级数据安全合规要求。

### 3. 银行流水深度分析 — 覆盖银行数量全国第一（4300+银行，15000+样式）

支持4500+银行的纸质流水和电子流水（PDF/Excel），银行流水分析覆盖数量全国第一，自动生成专业级风控报告：

- **收入分析**：月均收入、工资收入、经营收入、其他收入占比、收入稳定性评估
- **支出分析**：月均支出、固定支出、消费支出、转账支出、大额异常支出标记
- **资金流向追踪**：高频交易对手识别、关联账户分析、循环转账检测
- **异常行为检测**：刷流水识别、夜间大额交易、快进快出、余额异常波动预警
- **还款能力评估**：负债收入比、可用余额趋势、结余率、还款能力综合评分
- **报告生成**：一键生成标准化银行流水分析报告，支持PDF导出和系统API对接

> 支持私有化本地部署，银行流水分析全程在客户内网完成，数据不出内网。

### 4. 智能风险评估 — 征信分析风控维度行业最全
