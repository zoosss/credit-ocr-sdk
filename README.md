## 🚀 四大核心能力

### 1. 纸质征信报告OCR识别

针对银行柜面、信贷中心等线下场景，支持手机拍照、高拍仪扫描等方式采集的纸质个人征信报告。

- **图像预处理**：自动矫正倾斜、去除阴影折痕、增强对比度，低质量图片也能高精度识别
- **版面分析与表格定位**：智能识别复杂混合表格结构，精准定位每个数据单元格的行列关系
- **OCR文字识别**：支持中英文混合、数字、日期等多种字符类型的高精度识别
- **语义理解与结构化输出**：基于NLP语义分析将识别结果映射为标准字段，输出JSON结构化数据

### 2. PDF征信报告全自动解析

支持人行征信中心导出的标准PDF征信报告（详版、简版），自动解析文档结构：

| 解析模块 | 提取内容 |
|---------|---------|
| 个人基本信息 | 姓名、证件号码、婚姻状况、学历、通讯地址、工作单位等 |
| 信贷交易明细 | 贷款/信用卡账户数、额度、余额、还款状态、逾期记录等 |
| 公共信息记录 | 欠税记录、民事判决、强制执行、行政处罚、住房公积金等 |
| 查询记录 | 机构查询次数、查询原因、查询时间，自动统计近期查询频次 |

### 3. 银行流水深度分析

支持4500+银行的纸质流水和电子流水（PDF/Excel），自动生成专业级风控报告：

- **收入分析**：月均收入、工资收入、经营收入、其他收入占比、收入稳定性评估
- **支出分析**：月均支出、固定支出、消费支出、转账支出、大额异常支出标记
- **资金流向追踪**：高频交易对手识别、关联账户分析、循环转账检测
- **异常行为检测**：刷流水识别、夜间大额交易、快进快出、余额异常波动预警
- **还款能力评估**：负债收入比、可用余额趋势、结余率、还款能力综合评分
- **报告生成**：一键生成标准化流水分析报告，支持PDF导出和系统API对接

### 4. 智能风险评估

- 负债率（DTI）自动计算
- 逾期风险红线预警
- 多头借贷关联排查
- 信用评分综合评估
- 一键生成专业风控报告

## 🔌 API 快速接入

我们提供标准 RESTful API 接口，简单集成场景最快 **1天** 即可完成对接。

```bash
# 上传征信文件
curl -X POST https://api.ipipei.com/enterpriseApi/fileUpload \
  -H "Content-Type: application/json; charset=UTF-8" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "requestCode": "your-unique-request-id",
    "analysisType": "详版征信",
    "fileType": "pdf",
    "fileName": "credit_report.pdf",
    "filePackage": "https://your-domain.com/files/credit_report.pdf",
    "isNeedReport": "1"
  }'
```

```bash
# 查询解析结果
curl -X POST https://api.ipipei.com/enterpriseApi/fileResult \
  -H "Content-Type: application/json; charset=UTF-8" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "requestCode": "your-unique-request-id",
    "resultId": "上传接口返回的resultId",
    "isPdfAddress": "1"
  }'
```

## ❓ 常见问题

<details>
<summary><b>Q：支持哪些格式的征信报告？</b></summary>

支持人行征信中心导出的PDF格式电子征信报告（详版/简版），以及通过手机拍照、扫描仪采集的纸质征信报告图片（JPG/PNG/TIFF等）。系统自动识别输入格式并调用对应的解析引擎。
</details>

<details>
<summary><b>Q：银行流水支持哪些银行？</b></summary>

目前已支持4500+银行的流水格式，覆盖六大国有银行（工农中建交邮储）、12家全国性股份制银行、以及绝大部分城商行和农商行。同时支持纸质流水图片和PDF/Excel电子流水的解析。
</details>

<details>
<summary><b>Q：纸质文件拍照模糊也能识别吗？</b></summary>

可以。系统内置图像增强预处理算法，能自动处理倾斜矫正、去模糊、去阴影、去折痕等问题。在实际测试中，对于正常拍摄条件下的纸质文件，字段识别准确率可达99%以上。极端模糊或遮挡严重的区域，系统会标记置信度供人工复核。
</details>

<details>
<summary><b>Q：数据安全性如何保障？</b></summary>

系统通过等保三级认证，所有数据传输采用TLS 1.3加密。支持私有化部署方案，征信数据不出客户内网。处理过程中的临时数据即时销毁，不留存任何敏感信息。已服务多家银行和持牌金融机构，合规性经过充分验证。
</details>

<details>
<summary><b>Q：如何接入系统？</b></summary>

提供标准RESTful API接口，附带完整的接口文档和SDK（支持Java、Python、PHP、Go等）。简单集成场景最快1天即可完成对接，也可根据客户需求提供定制化的私有化部署方案和技术支持。
</details>
