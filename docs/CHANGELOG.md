# 更新日志

本文档记录拼图配征信识别平台的版本更新历史。

---

## [v2.3.0] - 2026-03-01

### 新增
- 新增 AI 征信分析报告功能（`isNeedReport` 参数），支持简版征信智能分析
- 新增企业使用额度查询接口 `enterpriseApi/fileFileCount`
- 银行流水模板新增 300+ 家城商行和农商行支持，总计覆盖 4500+ 银行

### 优化
- 纸质征信报告 OCR 识别准确率提升至 99.95%
- PDF 征信报告解析速度优化 40%，平均处理时间降至 20 秒以内
- 图像预处理算法升级，对严重倾斜和折痕的纸质文件识别效果显著提升

### 修复
- 修复部分银行流水 Excel 文件中合并单元格解析异常的问题
- 修复企业征信报告中特殊字符导致 JSON 输出格式错误的问题

---

## [v2.2.0] - 2025-12-15

### 新增
- 新增企业征信报告识别与解析
- 新增流水文件 Excel（xls/xlsx/csv）格式支持
- 新增 `actualFileType` 字段，自动检测客户上传文件类型是否匹配

### 优化
- 图神经网络（GNN）表格识别模型升级，复杂嵌套表格识别准确率提升 15%
- API 响应速度优化，平均延迟降低 30%

---

## [v2.1.0] - 2025-09-20

### 新增
- 新增纸质征信报告多张图片批量上传功能
- 新增 PDF 加密文件支持（`passWord` 参数）
- 新增报告 PDF 地址返回功能（`isPdfAddress` 参数）

### 优化
- 详版征信报告提取字段数扩展至 1050+
- 银行流水异常行为检测模型升级，刷流水识别准确率提升至 98%

---

## [v2.0.0] - 2025-06-01

### 新增
- 全新 RESTful API 架构上线
- 支持详版征信 / 简版征信 / 流水三大文件类型
- 银行流水深度分析功能上线（收支分析、资金流向、异常检测）
- Python / Java / PHP / Go SDK 发布

### 优化
- OCR 引擎全面升级为深度学习架构
- 系统架构迁移至分布式架构，支持高并发处理
