# 企业征信 OCR 识别分析案例 — 行业首家支持图片版+文字版双模式识别

> 本目录展示拼图配对 **企业征信报告** 的征信OCR识别与征信分析效果。支持私有化本地部署。

## 案例文件

| 文件 | 说明 |
|------|------|
| `企业征信（图片pdf版）_样例.pdf` | 原始企业征信报告样例（图片/扫描版） |
| `企业征信（图片pdf版）_样例-分析结果.pdf` | 图片版识别后的 AI 分析结果 |
| `企业征信（文字pdf版）_样例.pdf` | 原始企业征信报告样例（电子文字版） |
| `企业征信（文字pdf版）_样例-分析结果.pdf` | 文字版解析后的 AI 分析结果 |

## 支持能力

- 同时支持 **图片扫描版** 和 **电子文字版** 两种企业征信报告
- 自动提取企业基本信息、对外担保、借贷记录、诉讼信息等
- 企业征信文件大小限制 ≤ 1M
- 输出标准化 JSON 结构数据，可直接对接风控系统
- **支持私有化本地部署**，企业征信数据不出内网，满足银行级安全合规要求

## 如何使用

1. 下载两种版本的样例文件，对比不同输入格式
2. 下载分析结果文件查看结构化输出效果
